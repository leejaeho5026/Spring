
 Session Scope
=> Bean을 정의할 때, session scope로 정의하면(@SessionScope) => config  RootAppContext.java에서
   browser가 server에 최초로 요청(request)를 보낼 때
   Bean 객체가 주입됨
=> 주입만 이루어지지, session 영역에는 저장되지 않음

 Java 설정 방식에서는 @SessionScope 어노테이션 사용해서 설정함
(@SessionScope) => config  RootAppContext.java에서

 Xml 설정 방식에서는 bean을 정의할 때 scope="session"으로 설정함
 =>	   	<bean class="kr.co.tjoeun.beans.TestBeanByType" scope="session"></bean>
 =>	   	<bean class="kr.co.tjoeun.beans.TestBeanByName" id="sessionBean2" scope="session"></bean>
